# 🌐 Global Commodities Trading - React App

Aplicación web profesional desarrollada en **React + Vite + Tailwind CSS** para el sector de commodities internacionales.

## 📋 Características

- ⚡ **Vite** - Build tool ultrarrápido
- ⚛️ **React 18** - Última versión con Strict Mode
- 🎨 **Tailwind CSS** - Estilos utility-first
- 🌙 **Diseño Premium** - Paleta Industrial-Luxury (Dorado, Azul Marino)
- 📱 **Responsive** - Mobile-first approach
- 🔍 **SEO Ready** - Meta tags, Schema.org, Open Graph
- 👨‍💼 **EEAT Completo** - Perfil de autor, certificaciones, FAQ
- ♿ **Accesible** - ARIA labels, navegación por teclado

## 🚀 Instalación

```bash
# Clonar o descargar el proyecto
cd commodities-react

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Abrir en navegador
# http://localhost:3000
```

## 📦 Scripts disponibles

```bash
npm run dev      # Servidor de desarrollo (port 3000)
npm run build    # Build de producción
npm run preview  # Preview del build
npm run lint     # Linting con ESLint
```

## 🗂️ Estructura del proyecto

```
commodities-react/
├── index.html              # HTML principal con SEO meta tags
├── package.json            # Dependencias y scripts
├── tailwind.config.js      # Configuración Tailwind
├── vite.config.js          # Configuración Vite
├── postcss.config.js       # Configuración PostCSS
├── README.md               # Documentación
├── .gitignore              # Archivos ignorados
└── src/
    ├── main.jsx           # Punto de entrada React
    ├── App.jsx            # Componente principal (SPA)
    └── index.css          # Estilos globales + Tailwind
```
